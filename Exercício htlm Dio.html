Falando sobre Tags

(p) parágrafo

(blockquote) Essa tag você pode usar para fazer uma citação. Colocar algo de interessante sobre algum pensador. Isso é interessante, pois toda vez que eu qser fazer uma citação, posso colocar a tag <.blockquote.><./blockquote.> e colocar as palavras do pensador. Por exemplo, colocar as palavras sobre o pensamento de Eisntein depois de uma pesquisa completa dele.

(h2)aqui estou usando o (h2)

O h2 pode ser usado para colocar os subtítulos, esses são menores e deixam a página mais organizada. se quiser adicionar uma linha abaixo de cada 'h' voce pode ultilizar uma tag sem fechamento que é o 'hr'

Aqui usando minhas palavras somente para deixar tudo mais interessante para aprender html> Eu usei a tag strong para deixar a palavra interessante em negrito, e usei a tag 'i' para deixar a palavra 'negrito' em itálico. Eu estou a tag 'u' para deixar o nome sublinhado. Também usei a tag <.mark.> para deixar o texto marcado. Voce pode usar essas tags para chamar a atenção do leitor para uma palavra ou uma parte importante do seu texto. Uso da tag 'u' Vamos usar o 'mark' para usar como marcatexto* e deixar a oração 'Palavra interessante marcada'. Caso você queira colocar um asterisco, número, marcação de qualidade ou de referência, pode usar a tag <.sup.>*

Quer ver mais listas de tags voltadas para o texto, olha o site https://www.w3schools.com/ para colocar esse link diretório eu usei a tag 'a'.
